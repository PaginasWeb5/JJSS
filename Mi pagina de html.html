Descripción de CSS y HTML




HTML

HTML, sigla en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto), hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros. Es un estándar a cargo del World Wide Web Consortium (W3C) o Consorcio WWW, organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web, sobre todo en lo referente a su escritura e interpretación. Se considera el lenguaje web más importante siendo su invención crucial en la aparición, desarrollo y expansión de la World Wide Web (WWW). Es el estándar que se ha impuesto en la visualización de páginas web y es el que todos los navegadores actuales han adoptado.




CSS

Hojas de estilo en cascada (o CSS, siglas en inglés de Cascading Stylesheets) es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.2​ Es muy usado para establecer el diseño visual de los documentos web, e interfaces de usuario escritas en HTML o XHTML; el lenguaje puede ser aplicado a cualquier documento XML, incluyendo XHTML, SVG, XUL, RSS, etcétera. También permite aplicar estilos no visuales, como las hojas de estilo auditivas.

Etiquetas

html

Es el elemento raíz de la pagina, es lo que permite que exista.

head

Elemento que contiene información del documento.

title

Elemento que especifica el título del documento.

body

Elemento que contiene la parte visible de la pagina.

h1

Defina un encabezado largo, el numero puede varias dependiendo la importnacia siendo el 1 el mas importante y el 6 el menos importante.

p

Define un parrafo.

br

Se usa para dejar un espacio entre líneas, pero lo característico de esta es que a diferencia de las otras etiuetas no se cierra.

a

Significa "ancor" que en español seria ancla y refiere a que es un hipervínculo a un sitio, este tambien lleva lo que es un atributo llamado href que es donde se pone el link q se utilizara, tambien puede tener el atributo target. Ejemplo: https://www.google.com.mx/imghp?hl=es

img

Este elemento puede colocar una imagen, pero necesita de un atributo llamado src que significa source, aunque tambien se le pueden poner otros atributos como: alt para mostrar un texto si la imagen no carga, width y height para establecer el largo y el ancho.

table

Es una etiqueta que se utiliza para crear tablas.

caption

Esta etiqueta esta dentro de table y dicta el nombre de la tabla.

thead

Indica el la cabecera de la tabla, osea las columnas.

th

Indica el nombre y numero de columnas que tendra la tabla.

tr

Indica el numero de renglones que tendra la tabla.

td

Elemento que tiene el contenido de cada celda en los renglones por columna.

style

Este no es una etiqueta sino un atributo pero cabe mencionarlo porque entra dentro de muchas etiquetas permitiendoles cambiar el color, el tipo de letra, tamaño, alineación e incluso el fondo.
